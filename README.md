## 项目概述
```text
Spring Cloud 以两个库的形式提供了更多特性：Spring Cloud Context 和 Spring Cloud Commons
```

### Spring Cloud Context
```text
Spring Cloud Context 为 Spring Cloud 应用程序的 ApplicationContext（bootstrap context, encryption, refresh scope, 和 environment 端点）提供实用工具和特殊服务。 
```
### Spring Cloud Commons
```text
 Spring Cloud Commons 是在不同的 Spring Cloud 实现中使用的一组抽象和通用类（例如 Spring Cloud Netflix 和 Spring Cloud Consul）。
```
